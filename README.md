# 🏨 BookMyRooms – Lodge Management System

## 📌 Project Overview

BookMyRooms is a web-based Lodge Management System designed to simplify room booking and customer management for lodge owners. The platform allows users to browse available rooms and make bookings online, while admins can efficiently manage room details and customer records.

---

## 🚀 Features

### 👤 User Panel

* User Registration & Login
* Browse Available Rooms
* Book Rooms Online
* View Booking Details

### 🛠️ Admin Panel

* Admin Login
* Add / Update / Delete Room Details
* Manage Customer Information
* View and Manage Bookings

---

* **Backend:** PHP

## ⚙️ Installation & Setup

1. Clone the repository:

```id="bkmr2"
https://github.com/suvedant/BookMyRooms-Lodge-Management-System.git
```

2. Move the project to XAMPP htdocs folder:

```id="bkmr3"
C:\xampp\htdocs\
```

3. Start XAMPP:

* Apache ✔️
* MySQL ✔️

4. Import Database:

* Open phpMyAdmin
* Create a database (e.g., `bookmyrooms`)
* Import the `.sql` file

5. Run the project:

```id="bkmr4"
http://localhost/BookMyRooms/
```
